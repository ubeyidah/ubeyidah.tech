# Ubeyidah.tech — Portfolio

A minimal, fast personal portfolio showcasing projects, writing, and contact links.

## Live
- Website: https://ubeyidah.tech
- Blog: https://ubeyidah.tech/blog
- GitHub: https://github.com/ubeyidah

## Features
- Responsive hero, projects, and contact sections
- Blog list and article links
- Dark/light theme toggle
- SEO: sitemap and robots, structured data

## Tech Stack
- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- shadcn/ui, Radix UI
- Motion & small animation utilities
- Vercel Analytics

## Project Structure
- Pages: `app/page.tsx`, `app/blog/page.tsx`
- Layout: `app/layout.tsx`
- SEO: `app/sitemap.ts`, `app/robots.ts`
- Components: `components/*`
- Assets: `public/*`

## Getting Started
- Install: `npm install`
- Dev: `npm run dev` → open `http://localhost:3000`
- Build: `npm run build`
- Start: `npm run start`
- Lint: `npm run lint`

## Deployment
- Build with `next build` and deploy to Vercel (recommended) or any Node host.
